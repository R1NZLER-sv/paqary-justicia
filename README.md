# Paqary Justicia

Plataforma web interactiva para la visualización geográfica de las jurisdicciones
de los Jueces de Paz, con información de contacto y acceso a rutas mediante Google Maps.

## Tecnologías
- HTML, CSS, JavaScript
- Leaflet.js
- GeoJSON
- OpenStreetMap

## Funcionalidades
- Visualización de jurisdicciones por zonas
- Panel informativo por Juzgado de Paz
- Redirección a Google Maps
- Diseño responsive

## Estado del proyecto
Versión inicial (MVP). Proyecto escalable.
