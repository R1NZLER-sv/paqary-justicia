# Paqary Justicia

Plataforma web interactiva para la visualización geográfica de las jurisdicciones
de los Jueces de Paz, con información de contacto y acceso a rutas mediante Google Maps.

## Tecnologías
- HTML, CSS, JavaScript
- Leaflet.js
- GeoJSON
- OpenStreetMap

## Funcionalidades
- Visualización de jurisdicciones por zonas
- Panel informativo por Juzgado de Paz
- Redirección a Google Maps
- Diseño responsive

## Estado del proyecto
Versión inicial (MVP). Proyecto escalable.

## Criterio de delimitación territorial
Para la delimitación territorial de los Juzgados de Paz, el proyecto adopta como
criterio principal el límite distrital oficial, por ser la unidad político–administrativa
reconocida por el Estado Peruano. Las áreas geográficas representadas se construyen
a partir de cartografía pública y referencial, con fines informativos y de orientación
ciudadana, y no constituyen delimitación legal oficial.
